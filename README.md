Auto clicker made in .NET 10. 

Just open the .exe in the bin folder and while the console is open press F6 to work, will click every 5ms. 

You can change both the key and the interval changing the following variables: 

const int Hotkey -> To change the hotkey, you can following the link provided in the comment to see the virtual key codes. 

int clickInterval -> To change how faster the mouse click.
